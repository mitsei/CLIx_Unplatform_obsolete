********** Windows Instructions **********

Requirements:
Windows XP or higher
Chrome version 49+

******************** Installation ********************

1) Download and unzip to location of your choice


******************** Instructions ********************

1) Open the unzipped folder and locate unplatform_win32_ssl.bat

2) Click it to run. If any firewall permissions are requested, click allow.

3) Unplatform will launch in the background, then Chrome will open to https://localhost:8888  Refresh if necessary.

4) If prompted about an unsafe connection, click ADVANCED and then PROCEED. See instructions.png for reference. This should only need to be done once.

5) Leave the unplatform applications running in the background while browsing in Chrome

6) To exit unplatform, close the unplatform window.
